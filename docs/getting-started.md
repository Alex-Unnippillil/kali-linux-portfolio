# Getting Started

This project is built with [Next.js](https://nextjs.org/).

## Prerequisites

- Node.js 20
- yarn or npm

## Installation

```bash
yarn install
```

## Running in Development

```bash
yarn dev
```

See the [Architecture](./architecture.md) document for an overview of how the project is organized.

For app contributions, see the [New App Checklist](./new-app-checklist.md).

For UI strings and documentation updates, follow the [UI Microcopy Guidelines](./ui/microcopy.md) and run `yarn lint:microcopy` before opening a pull request.
