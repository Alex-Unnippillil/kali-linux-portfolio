Privacy & Diagnostics Simulations

The Kali Linux Portfolio runs entirely in your browser. The privacy dashboard lets you disable simulated access to capabilities so you can see how the desktop reacts when features are locked down.

What the toggles control

  • Telemetry stores high level usage counters in localStorage. No network requests are sent and you can wipe data by resetting settings.
  • Diagnostics enables extra logging inside developer tools and allows you to export anonymised JSON crash reports. These logs never leave the browser without you downloading them.
  • App permissions flip feature flags that the sandboxed apps use. For example the Security Tools app will disable clipboard buttons when access is revoked.

Storage locations

Settings are saved with IndexedDB or localStorage on your device. Clearing the browser storage or using the Reset option in the Settings app removes all collected data.
