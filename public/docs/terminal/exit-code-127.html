Exit code 127 — Command not found

The shell could not locate the command you entered. Either the package is not installed or the binary is not on your PATH.

Spot and fix typos

  • Re-run the command with tab completion to let the shell complete the name.
  • Compare to a known-good snippet in documentation or a cheat sheet.
  • Remember that command names are case-sensitive on Linux.

Install the missing tool

  • Search packages: apt-cache search <name> or apt search <name>.
  • Install: sudo apt install <package>.
  • Snap users: snap find and snap install.

Update your PATH

  • Add the directory with export PATH="/opt/tool/bin:$PATH".
  • Persist the change in ~/.bashrc or ~/.zshrc.
  • Restart the shell to apply the update.
