# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).


### Removed
- Removed all clipboard manager features and event listeners.

### Fixed
- Fixed Navbar overflow issue by stacking applications when limit is reached.
- Fixed Battleship chunk load error.
- Fixed Desktop preview functionality.
- Fixed Calendar visibility (SAT column) and crash on click.
- Fixed Battery dropdown status display.

### Improved
- Optimized folder layout for mobile and desktop.
- Updated Control Center with universal volume bar and improved icons.
- Modernized Settings app UI.
- Implemented "close on blur" for Whisker menu.

## [2.1.0] - 2025-09-03
### Added
- Added safe copy script and integrated into build process.

### Fixed
- Handle missing chat ID in chat module.
- Pin Vercel runtime version to prevent deployment errors.
- Update runtime mocks to resolve test issues.

### Upgrading
- No notable behavior changes.
